# 全球化的三角分工：负债国·生产国·资源国

> 全球化的经济格局由三个互相锁定、互相依赖的角色构成：负债国输出美元和消费需求，生产国输出制造能力和商品，资源国输出能源和原材料。美元是贯穿三者的血液。

## 三个角色

### 1. 负债国（消费国 / 央行国）

**代表：美国**

- **最大的消费市场**，通过贸易逆差吸收全球产出
- 输出**美元**——全球储备货币，其他国家赚美元后回流买美债
- 长期维持**经常账户赤字**，靠资本账户顺差（全球资本流入）平衡
- 经济驱动力：消费 + 金融，制造业占比持续下降

> 本质：用"信用"（美元 + 国债）换取全球实体商品和资源。

### 2. 生产国（制造国 / 加工国）

**代表：中国、日本、韩国、德国、东南亚（越南等）**

- 进口资源 + 加工制造 → 出口制成品
- 大量贸易顺差，积累巨额外汇储备（主要是美元资产）
- 经济驱动力：出口 + 投资，高度依赖外部需求
- 竞争力来源：劳动力成本优势 或 技术/效率优势

> 双重角色：① 从资源国买原材料；② 将成品卖给负债国。赚到的美元又借回给负债国。

### 3. 资源国（原材料国）

**代表：沙特、俄罗斯、澳大利亚、巴西、智利、加拿大（部分）**

- 输出能源、矿产、农产品等初级商品
- 收入高度依赖大宗商品价格周期
- 经常账户随商品价格波动：涨时盈余，跌时赤字
- 经济结构单一，工业化程度有限

> 把自然资源卖给生产国（和部分负债国），换回美元，形成"资源的美元计价"体系。

## 三角循环

```
资源国 ──原材料──→ 生产国 ──制成品──→ 负债国
   ↑                 ↑                   │
   │                 │                   │
   └──美元回流──────┘───美元回流──────────┘
       (买美债)         (买美债)      (输出美元+国债)
```

- **资源国** 把石油、矿产、农产品卖给生产国 → 赚美元 → 部分回流美债 / 主权基金
- **生产国** 进口原材料，加工制成品出口给负债国 → 贸易顺差赚美元 → 买美债（外汇储备本质上是"借条"）
- **负债国** 输出美元 + 国债 + 消费需求 → 贸易逆差（全世界为它生产） + 资本顺差（全世界借钱给它）

## 三个深层洞察

### 1. 美元环流

三种角色的共同分母是**美元**：
- 资源用美元计价
- 生产国赚美元
- 美元资产无处可去 → 回流美债
- 压低美国利率 → 继续支撑消费和杠杆

美元是贯穿三者的血液。

### 2. 不对等但相互锁定

表面上很不平等（美国印纸换实物），但三方都被锁死在体系内：

| 角色 | 锁死原因 |
|------|----------|
| 负债国 | 需要廉价商品 + 廉价融资 |
| 生产国 | 需要出口市场 + 美元资产的安全去处 |
| 资源国 | 需要稳定买家 + 美元计价体系 |

任何一方强行退出都会对自己造成巨大冲击。

### 3. 结构正在松动

- **负债国**：推动制造业回流和"再工业化"
- **生产国**：推动内需转型和货币国际化（如人民币）
- **资源国**：面临能源转型压力

这是 [[全球化-低利率-加杠杆的传导机制]] 所讨论框架可能瓦解的更深层原因。
