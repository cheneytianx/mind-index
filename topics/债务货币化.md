# 债务货币化

> 一句话定义：政府借钱，中央银行印钱买下这些债务——用印钞来消化财政赤字，把政府的欠条变成全社会的货币。

## 核心机制

### 正常发债 vs 债务货币化

```
正常财政融资：
  政府发国债 → 市场投资者购买（银行、基金、外国央行）
  → 市场资金减少（挤出效应）
  → 政府未来用税收还本付息
  → 钱量不变，只是换个口袋

债务货币化：
  政府发国债 → 央行购买
  → 央行凭空创造基础货币来支付
  → 市场中多了等量货币供给
  → 财政赤字 = 货币增发
```

**关键区别：** 正常发债是钱换个口袋，债务货币化是多了一笔钱。

### 资产负债表视角

**正常发债：**
```
        私人部门                          政府
  ┌──────────┬──────────┐    ┌──────────┬──────────┐
  │ 国债 +100 │ 存款 -100 │    │ 存款 +100 │ 国债 +100 │
  └──────────┴──────────┘    └──────────┴──────────┘
  资产结构变了，总量不变。
```

**债务货币化：**
```
        中央银行                          政府
  ┌──────────┬──────────┐    ┌──────────┬──────────┐
  │ 国债 +100 │准备金+100 │    │ 存款 +100 │ 国债 +100 │
  └──────────┴──────────┘    └──────────┴──────────┘
  货币总量增加了 100。政府债务变成了全社会的货币供给。
```

## 三种操作路径

### 1. 直接购买（最赤裸）

财政部发国债 → 央行在一级市场直接认购。

大多数发达国家法律禁止（如欧盟《马斯特里赫特条约》第 123 条）。战时经济和发展中国家历史上用过。

### 2. 二级市场购买：[[量化宽松]]的实质（最普遍）

```
国债在市场上发行 → 商业银行/投资者买下
     ↓
央行启动 QE，在二级市场大量购买国债
     ↓
国债进入央行资产负债表，银行体系多了准备金（基础货币）
```

**2008 年后美联储、欧央行、日本银行的核心操作。** 名义是货币政策，经济实质高度接近债务货币化。

日本走得最远：日本银行持有超过 50% 的日本国债，配合[[收益率曲线控制]]（YCC）人为将 10 年期国债收益率压在 0% 附近，市场定价机制被完全取代。

### 3. 财政透支（最隐蔽）

财政部在央行的账户直接透支，央行贷记——钱凭空产生，不需发债。

## 为什么这么做：[[财政主导]]

```
正常时期（货币主导）：
  央行设定利率 → 财政必须适应
  央行是主动方

财政主导（Fiscal Dominance）：
  政府债务太大，利率上升导致付息压力爆炸
  → 央行被迫压低利率（购债、YCC）
  → 货币政策服务于财政可持续性，而非物价稳定
```

**历史先例：** 1942-1951 年美国，美联储被迫维持低利率配合财政部融资，直到 1951 年 Fed-Treasury Accord 才重获独立性。

**当代案例：** 日本。国债/GDP 超 260%，利率升 1% 就能吃掉全部税收增长——日本银行本质上已进入[[财政主导]]状态。

## 后果与边界

### 通胀传导路径

```
央行购买国债 → 基础货币增加
     ↓
银行准备金过多 → 放贷能力增强
     ↓
如果实体经济有信贷需求 → 信贷扩张 → M2 增加
     ↓
如果供给跟不上 → 通货膨胀
```

**2008-2020 QE 未引发恶性通胀的原因：** 银行准备金大幅增加但信贷需求疲软——钱停在银行体系（超额准备金），水龙头没开。

**2021-2022 通胀爆发：** 财政直接发钱 + 供给瓶颈 = 水龙头开了，水管里的钱涌了出来。

### 货币化的边界

1. **通胀阈值：** 货币化超过实体经济吸收能力 → 通胀报复性上升
2. **汇率崩塌：** 全球市场对货币失去信心 → 货币化与贬值恶性循环（土耳其、阿根廷）
3. **主权信用丧失：** 极端情况无人愿持有该货币或债券——魏玛共和国、津巴布韦
4. **储备货币是护身符：** 美元作为全球储备货币，输出通胀的能力给了更大货币化空间，但并非无限

## 本质

**债务货币化就是把政府的欠条变成全社会的货币——用印钞机消化财政赤字。** 温和时叫 QE，是货币政策工具；过度时叫恶性通胀，是经济自杀。区别只在于：市场还信不信这个货币。
