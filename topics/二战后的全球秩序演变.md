# 二战后的全球秩序演变

> 二战后的秩序演变不是平滑的线性过程，而是一系列系统性断裂，每次断裂都重塑了游戏规则。

## 五次结构性断裂

### 第一阶段：1945-1971 | 布雷顿森林体系（规则约束型霸权）

核心安排：
- 美元挂钩黄金，其他货币挂钩美元 → 固定汇率体系
- 马歇尔计划 + 关贸总协定 → 西方经济一体化
- 联合国 + 布雷顿森林机构 → 美国提供全球[[公共品]]（安全、贸易规则、货币锚）

底层逻辑：美国以为盟友提供市场和安全来换取政治忠诚。这是一个开放性剥削——美国承受贸易逆差，换取体系主导权。

内在矛盾：[[特里芬难题]]——美元既要供应全球流动性（意味着逆差），又要维持黄金可兑换（要求顺差）。1971年尼克松关闭黄金窗口，[[布雷顿森林体系]]解体。

### 第二阶段：1971-1989 | 浮动汇率 + 新自由主义（权力转移型）

两次石油危机、越战失败、滞胀 → 旧框架崩溃。

关键转折：
- 1971年美元脱钩黄金 → 货币成为政治工具，汇率浮动 → 资本流动从受控变成自由
- 1979年沃尔克加息 → 利率飙升到20%，干掉通胀，但也引爆了拉美债务危机
- 1980年代里根-撒切尔改革 → 放松金融管制、私有化、贸易自由化 → 全球资本从"为生产服务"变成"为自己增殖"

深层逻辑：[[金融资本]]取代[[产业资本]]成为主导力量。美国不再需要贸易顺差来维持体系——印钞买你的商品，你赚了美元再买我的国债。这就是"超级特权"。

### 第三阶段：1991-2008 | 单极时刻（霸权型[[全球化]]）

苏联解体 → 美国成为唯一超级大国，"历史终结论"。

特征：
- [[华盛顿共识]] → 自由市场、开放资本账户、私有化输出到全球
- WTO成立（1995） → 全球供应链展开，中国加入成为最关键变量
- 科技泡沫 + 互联网革命 → 美国在IT/互联网领域的主导地位进一步巩固
- 9/11 → 反恐战争 → 美国的军事霸权开始过度延伸

本阶段底层矛盾：全球化在创造财富，但财富分配极度不公。美国中产阶级实际收入停滞，中国和德国等出口国积累了大量美元储备。

### 第四阶段：2008-2020 | 金融危机后的裂痕（霸权消耗期）

2008年金融危机暴露了两个东西：
- 金融化走到极端 → 金融从催化剂变成了定时炸弹
- 美国国内出了问题 → 救银行不救人的政治选择，催生了民粹主义

关键转折：
- 2010年欧债危机 → 暴露欧元区结构性缺陷
- 2013年中国提出"一带一路" → 首次有国家在全球供给替代性的秩序
- 2016年特朗普当选 + 英国脱欧 → 全球化的政治基础开始瓦解
- 2018年中美贸易战 → 新冷战在经济领域正式打响

深层逻辑：一个基于自由规则和开放市场的全球秩序，走到了其自身政治承载力的极限。当赢家太少、输家太多时，体系就会内爆。

### 第五阶段：2020-现在 | "多极"还是"无序"？（秩序重构期）

新冠 + 俄乌 + 中东滚雪球 → 加速了两个趋势：
- 脱钩断链 → 不是完全脱钩，而是"小院高墙"——AI、芯片、量子等关键领域搞阵营化
- [[去美元化]] → 不是替代美元，而是各国在对冲——增加其他货币和资产占比

当前可能的格局：
- [[G2格局]]（中美各自拉拢自己的体系）
- 或三极格局（美国+中国+全球南方试图独立）

最准确的描述：从单一中心秩序向[[多中心秩序]]的过渡期，而过渡期本身是最危险的——旧的规则被打破，新的尚未形成，所有人都按丛林法则行事。

## 核心动力链条

二战后的秩序演变有一个清晰的动力链条：

- **1945-1980：** [[产业资本]]主导（美国输出工业品+资本品）
- **1980-2008：** [[金融资本]]主导（美国输出美元+金融产品）
- **2008-现在：** [[地缘政治]]主导（美国输出安全+规则，中国输出基础设施+产能）

每次转移都是一次"秩序重构"，而每次重构的核心问题都是同一个：**谁来提供[[公共品]]？按照谁的规则？**

当前的问题在于：全球公共品严重不足——美国不愿或不能继续提供，中国尚不具备意愿和能力完全承接。这个真空期可能持续10-20年。
