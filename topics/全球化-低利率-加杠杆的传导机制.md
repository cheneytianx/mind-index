# 全球化 → 低利率 → 加杠杆的传导机制

> 全球化分工形成了不对等的资本循环，制造国赚美元、买美债，压低美国利率；美国企业借便宜钱加杠杆回购，推高资产价格。这个循环的脆弱性在于依赖全球化秩序的持续。

## 第一层：全球储蓄过剩 → 压低美国长期利率

全球化分工的本质是**生产向低成本地区转移，利润向资本方集中**：

- 制造国（中国、东南亚等）通过出口导向型经济积累大量贸易顺差，形成巨额外汇储备
- 这些储备需要配置**安全、高流动性**的资产——美国国债是几乎唯一的选择
- 全球过剩储蓄持续涌入美国债券市场 → 压低长期利率

### 债券价格与利率的反向关系

```
市场利率 ≈ 固定票息 / 债券价格
```

- 买盘 > 卖盘 → 债券价格被抬高 → 固定票息不变 → 实际收益率（利率）下降
- 这是一级市场的铁律，不是美联储单方面决定的

### 为何唯独美国国债？安全资产供需失衡

全球资本需求的是**大规模、高流动性、以储备货币计价的「安全」资产**：

| 选项 | 问题 |
|------|------|
| 欧元债券 | 欧元区非统一财政实体，各国信用分化 |
| 日本国债 | 国际化程度不够，市场深度有限 |
| 中国国债 | 资本管制，外资准入受限 |
| 黄金 | 规模不够，无利息 |
| **美国国债** | 唯一符合所有条件的「安全港」 |

> 核心结论：美国的长期利率不完全由本国经济决定，而是由全球资本供需格局决定。

## 第二层：低利率 → 企业加杠杆

### 结构性通胀抑制

- 制造业外迁 → 制成品价格持续走低
- 全球劳动力市场一体化 → 发达国家工人议价能力下降 → 工资增长受抑制
- 低通胀环境 → 美联储有条件维持宽松货币政策 → 政策利率中枢系统性下移

### 企业端激励

```
低融资成本 → 债务融资比股权更便宜
            → 企业倾向发债回购股票（而非扩大生产性投资）
            → EPS 提升 + 估值倍数扩张 = 股价上涨
```

- 典型现象：美国企业债膨胀 + 股票回购潮
- 资产负债率上升，但利率覆盖倍数（ICR）并不危险——因为利率太低

## 第三层：低利率 → 估值抬升的自反馈

```
低利率 → DCF 模型折现率降低 → 未来现金流现值增加 → 估值抬升
       → 投资者追逐收益率（reach for yield）→ 资金涌入风险资产
       → 上市公司便宜融资 → 加杠杆回购 → 进一步推高股价
```

## 本质性洞察

全球化构建了一个**不对等的循环**：

- **生产端**（发展中国家）：提供廉价劳动力和商品
- **金融端**（发达国家）：提供资本市场和储备资产
- 制造国赚美元 → 买美债 → 压低美国利率 → 美国企业借便宜钱做金融套利 → 推高资产价格

这个循环的脆弱之处在于依赖全球化秩序的持续。贸易摩擦、供应链重构、制造业回流等趋势一旦逆转，低利率的基础就可能动摇——过去几年的高通胀 + 加息，正是这个结构松动的表现。

## 结构正在改变？

过去几年出现重要变化：外国持有美债的占比在下降，取而代之的是美国家庭和美联储自身。整个全球储蓄循环可能正在重组。
